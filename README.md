## 🛠️ Getting Started

1. **Clone the repository:**
   ```
   git clone https://github.com/kanchan191184/Kanchan_portfolio.git
   ```
2. **Install sass:**
   ```
   npm install -g sass
   ```

3. **Compile Sass:**
   ```
   sass --watch styles.scss style.css
   ```
